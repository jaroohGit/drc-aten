# แนะนำการใช้งาน DRC Online

## ขั้นตอนการติดตั้ง

### สำหรับผู้ใช้งานครั้งแรก

1. **ดาวน์โหลดและแตกไฟล์**
   - แตกไฟล์ทั้งหมดไปยังโฟลเดอร์ที่ต้องการ

2. **ติดตั้งโปรแกรม**
   - ดับเบิลคลิก `install.bat`
   - รอให้การติดตั้งเสร็จสิ้น
   - จะสร้างไฟล์ `start_drc_online.bat` อัตโนมัติ

3. **เชื่อมต่ออุปกรณ์**
   - เสียบ NanoVNA เข้ากับ USB
   - ตรวจสอบหมายเลข COM Port ใน Device Manager
   - แก้ไขหมายเลข COM Port ในไฟล์ `app.py` (ถ้าจำเป็น)

4. **เริ่มใช้งาน**
   - ดับเบิลคลิก `start_drc_online.bat`
   - เปิดเว็บเบราว์เซอร์ไปที่ http://localhost:5000
   - คลิกปุ่ม "▶ Start Sweep" เพื่อเริ่มวัดค่า

---

## วิธีแก้ไขการตั้งค่า

### เปลี่ยนหมายเลข COM Port

1. เปิดไฟล์ `app.py` ด้วย Notepad
2. หาบรรทัด `PORT = "COM4"`
3. เปลี่ยนเป็นหมายเลข COM Port ของคุณ เช่น `PORT = "COM3"`
4. บันทึกไฟล์

### เปลี่ยนช่วงความถี่

1. เปิดไฟล์ `app.py`
2. หาบรรทัด:
   ```python
   START_FREQ = 4.40e9  # 4.40 GHz
   STOP_FREQ = 5.00e9   # 5.00 GHz
   ```
3. แก้ไขค่าตามต้องการ
4. บันทึกไฟล์

### เปลี่ยนความเร็วอัพเดท

1. เปิดไฟล์ `app.py`
2. หาบรรทัด `INTERVAL = 0.2  # seconds (200 ms)`
3. เปลี่ยนค่า เช่น:
   - `0.1` = 100 ms (เร็วขึ้น)
   - `0.5` = 500 ms (ช้าลง)
   - `1.0` = 1000 ms (1 วินาที)
4. บันทึกไฟล์

## การแก้ปัญหา

### ไม่พบอุปกรณ์
- ตรวจสอบสาย USB
- ลงไดรเวอร์ Cypress USB-CDC (สำหรับ Windows 10 และเก่ากว่า)
- ลองเสียบ USB port อื่น

### ไม่สามารถเริ่มโปรแกรมได้
- ตรวจสอบว่าติดตั้งเรียบร้อยแล้วโดยรัน `install.bat`
- ตรวจสอบว่ามี Python ติดตั้งอยู่
- ลองรัน Command Prompt แบบ Administrator

### กราฟไม่อัพเดท
- คลิกปุ่ม Stop แล้วคลิก Start ใหม่
- รีเฟรชหน้าเว็บ (กด F5)
- ตรวจสอบว่าอุปกรณ์เชื่อมต่ออยู่

---

## คุณสมบัติโปรแกรม

### กราฟแสดงผลแบบเรียลไทม์
- S11 Magnitude (dB)
- S22 Magnitude (dB)
- S11 Phase (องศา)

### ข้อมูลสถิติ
- ค่าเฉลี่ย (Average)
- ค่าสูงสุด (Maximum)
- ค่าต่ำสุด (Minimum)

### การอัพเดท
- อัพเดทอัตโนมัติทุก 200 มิลลิวินาที
- แสดงจำนวนครั้งที่วัด
- แสดงเวลาอัพเดทล่าสุด

---

## การนำไปติดตั้งเครื่องอื่น

1. คัดลอกโฟลเดอร์ `drc-aten` ทั้งหมด
2. วางไว้ในเครื่องคอมพิวเตอร์ใหม่
3. รัน `install.bat` บนเครื่องใหม่
4. แก้ไข COM Port ในไฟล์ `app.py` (ถ้าจำเป็น)
5. รัน `start_drc_online.bat`

**หมายเหตุ**: เครื่องปลายทางต้องมี Python 3.8 หรือสูงกว่าติดตั้งอยู่
